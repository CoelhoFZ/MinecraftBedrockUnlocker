Por que escolher o Unlocker?

Desenvolvido para ser simples, rápido e transparente.

Desbloqueio Total

Remove todas as limitações do modo Trial. Jogue sem limite de tempo, acesse servidores e use skins personalizadas.

Open Source

Como Instalar (Passo a Passo)

Siga rigorosamente para garantir o funcionamento.

1

Instalar via Xbox App

Baixe o "Minecraft for Windows" (Trial) apenas pelo aplicativo Xbox. Não use a Microsoft Store diretamente.

2

Baixar o Unlocker

Faça o download da última versão (mc_unlocker.exe) na nossa página de releases.

3

Executar como Admin

Clique com botão direito no arquivo e selecione "Executar como Administrador". É necessário para modificar a pasta do jogo.

4

Ativar o Mod

No menu do programa, digite 1 e pressione Enter. Aguarde a mensagem de sucesso.

Alerta de Antivírus

Como esta ferramenta modifica arquivos do jogo (injeção de DLL), é comum que antivírus detectem como "HackTool" ou falso positivo. Recomendamos adicionar uma exceção para o arquivo ou desativar temporariamente a proteção em tempo real.
